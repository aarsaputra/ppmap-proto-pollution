Security Tiers

Tier 0-2: Standard Detection

Tier 3-5: Advanced Gadgets

Focuses on DOM execution sinks like innerHTML, eval(), and setTimeout(). Also tests server-side Node.js environments and Template Engine (EJS/Handlebars) pollution.

Tier 6-8: API & Protocol Specialists
