Usage & Commands

Master the PPMAP CLI with advanced scanning techniques.

Basic Scanning

Run a full scan with all 9 detection tiers against a single target.

$ python3 ppmap.py --scan "https://example.com"

Advanced Flags

Flag Description
--poc URL Run only jQuery-specific Quick PoC tests.
--oob Enable Out-of-Band (OOB) detection for blind vulnerabilities.
--stealth Enable anti-bot/WAF evasion measures (jitter, custom headers).
--workers N Set number of concurrent scanner threads (default: 3).
--discovery Enable deep endpoint discovery via JS regex matching.
--format FMT Output format: json, html, markdown, csv.

💡 Security Best Practices

  • 🎯 Targeting: Always use --poc first to check for basic jQuery vulnerabilities before running a deep --scan.
  • 🕵️ Stealth: When scanning via VPN or Proxy, use --stealth --delay 1 to avoid rate-limiting.
  • 📊 Reporting: Use --format json,html to get both machine-readable and human-friendly results.
  • 🚀 Performance: PPMAP v4.1.0 automatically skip static files. For large sites, use --workers 5.
