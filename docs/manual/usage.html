Usage & Commands

Master the PPMAP CLI with advanced scanning techniques, pipeline integrations, and enterprise configurations.

Basic Scanning

# 1. Full Assessment (Discovery + Deep Scan)
$ python3 ppmap.py --scan-full "https://example.com"
# 2. Targeted Scan (Provided URL only)
$ python3 ppmap.py --scan "https://example.com/api/v1"
# 3. Discovery Only (Recon Mode)
$ python3 ppmap.py --discover "https://example.com" --max-depth 2

Scanning Options Reference

Flag Description Example
--scan URL Start a full scan (all tiers 0-7) on target(s) --scan https://target.com
--poc URL Run Quick PoC (jQuery only) --poc https://target.com
-ls, --list FILE Scan targets from a file (one URL per line) -ls targets.txt
--stdin Read targets from pipe (e.g., from subfinder) cat urls.txt | python3 ppmap.py --scan --stdin
-r, --request FILE Scan request from a file (Burp Suite format) -r req.txt

Browser & Performance

Flag Description Default
--browser {chrome,firefox} Browser engine to use chrome
--headless Run browser in background (no UI) True
--no-headless Show browser window (good for debugging) False
--workers N Number of concurrent workers (threads) 3
--timeout N Request timeout in seconds 30
--async-scan Enable experimental async engine False

Stealth & Evasion Modules

Flag Description
--stealth Enable anti-bot/WAF evasion mode
--delay N Delay between requests in seconds (e.g., --delay 2)
--user-agent STR Custom User-Agent string
--proxy URL Use HTTP/S proxy (e.g., --proxy http://127.0.0.1:8080)
--oob Enable OOB/Blind detection (Interact.sh)
--disable-xss Disable XSS tests and focus only on PP object injection
--disable-waf-bypass Disable WAF bypass tests to save time
--insecure Disable SSL certificate verification

Reporting & Utility

Flag Description
--output DIR Base directory for reports (default: reports/). Reports are auto-organized.
--format FMT Output formats: json, html, markdown, csv, xml
--template TPL HTML template styles: modern (default), minimal, or detailed
--diff FILE1 FILE2 Compare two scan result files (delta reporting)
--no-poc Exclude Proof-of-Concept HTML/Code strings from report (safe mode)

Enterprise Scenarios

PPMAP is built for scale. Here are real-world deployment examples.

1. Bulk Bug Bounty Hunting (Mass Scanning)

$ subfinder -d example.com -silent | httpx | python3 ppmap.py --scan --stdin --headless --workers 10

2. Deep Forensic Audit (SAST + DAST)

# 1. First, scan the source code (SAST)
$ python3 -m ppmap.sast --dir ./src --output sast_findings.json
# 2. Then, verify with browser automation (DAST)
$ python3 ppmap.py --scan "https://staging.target.com" --oob --async-scan
