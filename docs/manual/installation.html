Installation Guide

PPMAP is built for portability and research. Follow the platform-specific instructions below.

Quick Jump Menu

  • 🐧 Linux (Kali/Arch/Ubuntu)
  • 🪟 Windows (WSL2)
  • 🍎 macOS
  • 🐳 Docker

📋 Standard Requirements

  • ✅ Python: v3.8 or higher (v3.11+ recommended)
  • ✅ Browser: Chrome, Chromium, or Brave installed
  • ✅ Drivers: Handled automatically by webdriver-manager
  • ✅ OS: Linux (Kali/Ubuntu), macOS, Windows (WSL2)

Linux (Kali / Arch / Ubuntu)

We recommend using venv to keep your system clean. On modern Kali/Debian distributions, avoid global pip installations (PEP 668).

Debian-based (Kali/Ubuntu)

$ python3 ppmap.py --scan-full http://localhost:3000

Arch Linux / BlackArch

$ sudo pacman -Sy git python chromium
$ git clone https://github.com/aarsaputra/ppmap-proto-pollution.git
$ cd ppmap-proto-pollution && python3 -m venv venv
$ source venv/bin/activate && pip install -r requirements.txt

Windows (PowerShell / WSL2)

We highly recommend using WSL2 for the best experience, but native PowerShell is also supported.

Option A: WSL2 (Recommended)

Follow the Linux instructions inside your WSL terminal.

$ sudo apt update && sudo apt install git python3-venv chromium-browser -y

Option B: PowerShell (Native)

PS> git clone https://github.com/aarsaputra/ppmap-proto-pollution.git
PS> cd ppmap-proto-pollution
PS> pip install -r requirements.txt

macOS (Homebrew)

$ brew install python chromium
$ git clone https://github.com/aarsaputra/ppmap-proto-pollution.git
$ cd ppmap-proto-pollution && python3 -m venv venv
$ source venv/bin/activate && pip install -r requirements.txt

Docker (Sandboxed)

Perfect for one-off scans or running in isolated environments.

# Build the image
$ docker build -t ppmap .
# Run a scan
$ docker run --rm ppmap --scan-full http://target.com

✅ Verify Installation

$ python3 ppmap.py --help

If you see the help menu with all 32 scanning options, you are ready to go!

⚠️ Troubleshooting

1. error: EXTERNALLY-MANAGED-ENVIRONMENT (Kali/Ubuntu 24+)

Modern Debian versions block global pip installations by design. You MUST use a Python virtual environment (python3 -m venv venv) before running pip install -r requirements.txt.

2. Chrome/Chromium not found

Ensure a browser is installed and accessible in your PATH. On Linux/WSL, install chromium-browser.

3. SSL: CERTIFICATE_VERIFY_FAILED

Run with --no-verify-ssl if you are testing targets with self-signed certificates.
