Installation Guide

PPMAP requires Python 3.8+ and a Chromium-based browser for full automation.

📋 Standard Requirements

  • ✅ Python: v3.8 or higher (v3.11+ recommended)
  • ✅ Browser: Google Chrome, Chromium, or Brave
  • ✅ RAM: 2GB minimum (4GB recommended for heavy async scans)
  • ✅ OS: Linux, macOS, or Windows (via WSL2 or PowerShell)

Linux (Kali / Ubuntu)

# 1. Update system
$ sudo apt update && sudo apt install git python3-pip chrome-driver -y
# 2. Clone and install
$ git clone https://github.com/aarsaputra/ppmap-proto-pollution.git
$ cd ppmap-proto-pollution
$ pip install -r requirements.txt

Windows (PowerShell / WSL2)

We highly recommend using **WSL2** for the best experience, but native PowerShell is also supported.

Option A: PowerShell (Native)

PS> git clone https://github.com/aarsaputra/ppmap-proto-pollution.git
PS> cd ppmap-proto-pollution
PS> pip install -r requirements.txt

Option B: WSL2 (Recommended)

Follow the Linux instructions inside your WSL terminal.

$ sudo apt update && sudo apt install python3-pip chromium-browser -y
